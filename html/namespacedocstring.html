This is the location of all of the checks/tests used in the project. The base check class does little beyond provide an outline that should be followed when implementing checks.

This module exists because the boxcox class in stastmodels is not particularly useful. It's output must be manipulated to actually be used in a pratical setting. This module does that manipulation.

This is the location of all of the visualizations used in the project. There was enough variety in the visualizations, that creating a base class seemed unecessary, however that may be an area for expansion in the future.
